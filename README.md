# 🇪🇪 Pronoun Olema Drill

Хардкор-тренажёр эстонских местоимений, глагола olema и профессий.

**[▶ Играть](https://vicso-name.github.io/pronoun-olema-drill/)**

## Что тренируем

| Местоимение | Olema | Отрицание |
|-------------|-------|-----------|
| ma (mina) — я | olen | ei ole |
| sa (sina) — ты | oled | ei ole |
| ta (tema) — он/она | on | ei ole |
| me (meie) — мы | oleme | ei ole |
| te (teie) — вы | olete | ei ole |
| nad (nemad) — они | on | ei ole |

Профессии: arst, õpetaja, programmeerija, õpilane (+ множественное число)

## 5 типов упражнений

1. **Выбери форму / местоимение** — `Ma ___ arst` → olen/oled/on/oleme
2. **Собери предложение** — слова вразброс + обманки
3. **Трансформация** — утверждение ↔ отрицание
4. **Полный перевод** — RU→ET, ввод вручную
5. **Аудио-диктант** — послушай и напиши предложение

## Фичи

- **SM-2 Spaced Repetition** — интервальное повторение между сессиями
- **Прогрессивная сложность** — 80% выбор в начале → 70% ввод к концу
- **72 озвученных предложения** (Google Cloud TTS Chirp3-HD)
- **Авто-озвучка** правильного ответа после каждого вопроса
- Адаптивный трекинг 18 навыков (6 местоимений × 3 формы)
- Сохранение прогресса (localStorage)
- 50 вопросов за сессию
